# Исходники контента (черновик → прод)

Структура из **Недели 1** роадмапа (`my-guide/COMPOSER-2-MASTER-PROMPT.md` §7):

- `professions/` — тексты `/personal/*` (после вычитки из `my-guide/content/professions/`)
- `industries/` — отрасли (`/otrasli/*`)
- `services/` — услуги (`/uslugi/*`)
- `locations/` — география
- `blog/` — статьи (или синхрон с `src/content/blog-stub`)

Пока каталоги пустые: страницы собирают метаданные из `src/lib/site-structure.ts`, полный копирайт подключается по мере появления `.md` файлов.
