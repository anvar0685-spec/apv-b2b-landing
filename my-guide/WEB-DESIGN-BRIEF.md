# Бриф веб-дизайна (Неделя 1)

Зафиксировано ассистентом как **рабочие допущения v1** (2026-04). Если хочешь заменить референсы или табу — правь блоки 1–2 и дату внизу.

## 1. Три референса (URL + одно предложение «что взять»)

1. **https://linear.app** — плотная типографика, тёмная сцена без «кричащих» градиентов, быстрые микро-состояния; ориентир на **ясность и технологичность** без перегруза декора.
2. **https://stripe.com** — ритм длинных B2B-страниц: **сетка, воздух, доверие через структуру**, а не через стоковые фото; таблицы и блоки сравнения как образец для тарифов/SLA.
3. **https://selectel.ru** — **РФ B2B-язык**: спокойный корпоративный тон, длинные посадочные без визуального шума, ощущение «серьёзного подрядчика» (не агентство «креатив ради креатива»).

## 2. Три табу («так не делаем»)

1. **Стоковые «счастливые люди в касках»** и любой визуал, который выглядит как банковский/HR-шаблон 2015 года — снижает доверие у директора склада.
2. **Неон, «крипто»-градиенты, glassmorphism ради эффекта** на первом экране — не соответствует нише складской логистики и **бьёт по LCP/CLS**.
3. **Перегруженный первый экран** (слишком много CTA, всплывающих виджетов, бегущих строк одновременно) — B2B-решение принимается **спокойно**, не как в игре.

## 3. Доминанта темы на коммерческих страницах

- [ ] преимущественно светлая  
- [ ] преимущественно тёмная  
- [x] **смешанная** (как сейчас в коде: **тёмный hero-band** + **светлое тело** + опционально тёмные секции внизу страницы)

Комментарий: для **калькулятора и длинного чтения** (блог) приоритет **светлый фон**; для **hero и «операционных» блоков** допустима тёмная сцена как у главной — сохраняем **единый ритм** «тёмное вверху → светлая мясная часть → CTA».

## 4. Акцентный цвет

- [x] **оставляем текущий teal** (`--accent` / `--accent-soft` в `globals.css` и `theme-default.ts`) как основной сигнал «действие / ссылка / фокус».
- [ ] меняем (опиши RAL/HEX или словами): *не выбрано*

**Допущение v1:** второй акцент **не вводим**, пока не появится брендбук; при необходимости выделить **один** тип CTA (например «Позвонить») можно позже точечно добавить **янтарный** (`#f59e0b` из мастер-дока) **только для primary-кнопки**, не меняя глобальный `--accent`.

## 5. Персона №1 (одна фраза)

**Операционный директор / директор склада в Москве или МО**, которому нужен **предсказуемый поставщик смен** (явка, замены, цифры, compliance), а не «ещё один кадровый агрегатор».

## 6. Фото / видео

Что можем публиковать с правами (отметь):

- [x] фото склада — **цель**: 1–3 реальных кадра с объектов (без людей в кадре или с подписанными релизами); **до съёмки** — абстрактные схемы/градиенты как сейчас.
- [x] фото команды — **цель**: 1 групповой / 2 портрета руководства; **до съёмки** — не использовать сток с лицами.
- [x] логотипы клиентов — **только с письменного разрешения**; до этого — нейтральные «кейсы без логотипа» или монохромные плейсхолдеры без подделки брендов.
- [ ] видео со объекта — **опционально** этап 2; не блокирует запуск дизайна.
