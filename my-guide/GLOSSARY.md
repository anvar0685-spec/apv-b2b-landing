# Глоссарий (черновик Недели 1)

Редактируй по мере согласования с копирайтом. Цель — единый язык на всех URL.

| Термин | Пишем так | Не пишем / комментарий |
|--------|-----------|-------------------------|
| Услуга | аутсорсинг (складского) персонала | аутстаффинг как отдельный продукт (см. мастер-док §0) |
| Локация | Москва и Московская область / МО | «по всей России» без уточнения |
| Смена | смена, график, выход на объект | разговорные сокращения без определения |
| Ставка | от X ₽/час (после согласования цифр) | «низкая цена», «дешёво» |
| Документы | compliance, регламенты | пустые обещания без процесса |

## Запрещённые фразы (напоминание §8.2)

- «качественный аутсорсинг», «профессиональный подход», «индивидуальный подход», «широкий спектр услуг» — заменять на факты и цифры.
