### Не сделано или не закрыто без заказчика (не врать новому агенту)

| Что | Почему |
|-----|--------|
| **Финальные тарифы, юридически утверждённые цифры, медиа** | В текстах — ориентиры и редакционные заготовки; КП/договор — вне сайта. |
| **Уникальные лонгриды на все programmatic URL** | Шаблон + приоритетные 30; остальные ~300 пар — базовый шаблон. |
| **§5 мастер-дока 1:1** | Часть URL из мастер-дока (другие slug услуг) может не совпадать с текущей витриной — сверять `COMPOSER-2-MASTER-PROMPT.md`. |
| **Прод API лидов / env** | Роут есть; боевой конфиг и проверка — отдельно. |
| **Недели 7–8** | Калькулятор по §7, кейсы — см. таблицу ниже, не входили в «6 недель». |

### Куда смотреть в первую очередь

- План: **`SIX-WEEK-PLAN.md`**, мастер: **`COMPOSER-2-MASTER-PROMPT.md`**, прогресс (этот файл).
- Правила агента: **`apv-b2b-landing/.cursor/rules/`** (`00` … `03`).
- Сборка: из **`apv-b2b-landing/`** → **`npm run build`**.
